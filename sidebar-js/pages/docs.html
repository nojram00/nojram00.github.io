This is the documentation part for sidebar js:
To Begin, you need to copy these following tags inside the <head> tag: <script src="https://nojram00.github.io/dist/sidebar.min.js"></script> <link rel="stylesheet" href="https://nojram00.github.io/dist/sidebar.min.css"> Then, create a <sidebar> tag. This serve as a main container and a sidebar element <sidebar>
   <!-- You can add an elements inside or a navigation bar if you want. -->
   <div>
    
   </div>
</sidebar>
To create a toggle, simply add a 'toggle' attribute inside the element. For example: a button. <button toggle> Toggle Me! <button> To create a toggle that only open the sidebar, use 'toggle-open' attribute. <button toggle-open> Toggle Me! <button> To create a toggle that only close the sidebar, use 'toggle-close' attribute. <button toggle-close> Toggle Me! <button>
